## Purpose

### 1. 우테코 합격

우테코 합격으로 인해 , 자바를 공부하게 되었습니다.
백엔드 커리큘럼 중 모던 인 자바 액션이 있으나 , 
바로 공부하기에는 기본 지식 및 자바에 대한 내용을 요구하므로
유투브를 보며 코드 와 함께 사전 공부하려고 합니다.

### 2. 함수형 프로그래밍

Typescript 에도 함수형을 지원하는 라이브러리 와 기능들이 있으나,
실제로 사용을 하지 않았습니다.
코드를 간결하게 + 병렬적 + Pure 하게 해주는 함수형 프로그래밍에 대해
이번 기회에 공부해볼 예정입니다.
